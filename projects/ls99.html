Problem DARI PROJECT - DETAIL - LS099

Brand belum konsisten; e-commerce tanpa tracking memadai. Target: trust visual & pengukuran funnel end-to-end.

Goals

  • G1 — +5% checkout conversion
  • G2 — Implement GA4 enhanced ecommerce
  • G3 — Retention cohort 90 hari

Approach & Architecture

  1. Design system (token warna/typo) → konsistensi across assets.
  2. Checkout UX: trust badges, shipping clarity, sticky CTA.
  3. GA4 EE + GTM server-side untuk event yang andal.

Results

  • +8% CVR; CTR PDP → cart +13%.
  • Retensi 90 hari naik 6pt pasca bundle.
  • Dashboard cohort mingguan untuk marketing.

Learnings

  • SSOT event naming memudahkan analisis lintas kampanye.
  • Packaging konsisten menaikkan trust di PDP.
  • Bundling diskon generik untuk LTV.

Responsibilities

  • Brand system (logo, color, packaging dieline)
  • Shopify theme setup & conversion best practices
  • GA4 enhanced ecommerce + server events
  • Retention cohort dashboard
